Branding | Plugin Documentation | Translating Exodus | Compiling Source
  1. Install Borland Developer Studio 2006.
  2. Install the Jedi Code Library from here. Use at least release 1.97 for BDS2006.
  3. Install and setup Subversion.
  4. Checkout the Exodus source code
    • If you have an ssh account on jabberstudio.org, use:

      svn co svn+ssh://username@svn.jabberstudio.org/var/projects/exodus/svn/trunk exodus

    • If you want to do anonymous checkouts, use:

      svn co svn://svn.jabberstudio.org/exodus/svn/trunk exodus

  5. Unzip tntUnicode.zip from the Exodus\Exodus\Components directory.
  6. Open Exodus\Exodus\Components\tntUnicode\Packages\BDS4\TntUnicodeVCL_Design.bdsproj
  7. From the Project Manager, Right click, and select Install.
  8. Open Exodus\Exodus\Components\ExComponents.dpk
  9. From the Project Manager, Right click, and select Install.
  10. Open Exodus\Exodus.dpr
  11. Click "Build Exodus"
