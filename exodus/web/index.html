If you're interested in working on the development of Exodus, please feel free to checkout the source code. Make sure to the read the readme.txt located in the main exodus directory before emailing me about compilation issues.
Using SSL Connections
To use SSL connections, download the SSL DLL's. This zip package contains 2 DLL's which you must place into the same directory as you installed Exodus into. These DLL's are not included by default to keep the initial download size of Exodus down.
Are WEIRD things happening in your chat windows?
If you are seeing lots of strange text instead of emoticons, an empty text-box control for changing the properties for chatrooms in Options -> Fonts, Colors or if the text-box controls in your chat- windows don't update itself after getting or sending text, you probably need to update your system's RichEdit control. Please download richupd.exe which is an update from Microsoft. It should update your system. During this process make sure Exodus is closed. Restart your computer (if necessary) then restart Exodus. You should now have full featured chat windows (including emoticons) and TC rooms.
