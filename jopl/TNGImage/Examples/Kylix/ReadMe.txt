TNGImage for Kylix
==================

This directory contains examples for Kylix.
(They were only tested with K1, but should work for any later versions)

The following shared lib's (next to Kylix' own requirements) are required to
make the examples work:

libz.so (1.1.3)
libjpeg.so (6.2)
libmng.so (1.0.0+)
